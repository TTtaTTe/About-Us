# 📦 보고서 다관점 리뷰 에이전트 - 설치 가이드

## 파일 구성

| 파일 | 설명 |
|------|------|
| `report-reviewer.md` | 에이전트 본체 (이것만 있으면 됩니다) |

---

## 설치 방법 (3단계)

### 1단계: 폴더 찾기

Windows 기준:
```
C:\Users\내사용자이름\.claude\agents\
```

Mac 기준:
```
/Users/내사용자이름/.claude/agents/
```

> 💡 `.claude` 폴더가 안 보이면?
> - Windows: 파일 탐색기 상단 → 보기 → "숨긴 항목" 체크
> - Mac: Finder에서 Cmd + Shift + . 누르기
>
> 💡 `agents` 폴더가 없으면?
> - `.claude` 폴더 안에 `agents` 폴더를 새로 만드세요
>
> 💡 `.claude` 폴더조차 없으면?
> - Claude Code를 한 번이라도 실행했다면 자동으로 생깁니다
> - 없으면 직접 `.claude` 폴더를 만들고, 그 안에 `agents` 폴더를 만드세요

### 2단계: 파일 복사

`report-reviewer.md` 파일을 위 폴더에 복사합니다.

최종 경로 예시 (Windows):
```
C:\Users\hong\.claude\agents\report-reviewer.md
```

### 3단계: Claude Code에서 확인

Claude Code (터미널 또는 데스크탑)를 열고 `/agents`를 입력하면
`report-reviewer`가 목록에 보이면 설치 완료!

---

## 사용 방법

### 기본 사용
Claude Code에서 이렇게 말하면 됩니다:

```
보고서 리뷰해줘
```
그리고 리뷰할 파일을 함께 전달하세요.

### 파일 경로 지정
```
/path/to/보고서.md 이 보고서 리뷰해줘
```

### 옵션 변경
```
보고서 리뷰해줘 통과점수 8점으로
보고서 리뷰 5라운드까지 해줘
전략가 관점만 리뷰해줘
```

---

## 리뷰 프로세스 요약

```
📄 문서 입력
    ↓
🔷 전략가 리뷰 → 점수
🔷 분석가 리뷰 → 점수
🔷 독자대변인 리뷰 → 점수
    ↓
📊 전원 7점 이상? ─── Yes ──→ ✅ 완료, 파일 저장
    │ No
    ↓
🔄 피드백 반영 후 다음 라운드
    (최대 3라운드)
    ↓
📄 개선된 문서 저장 (원본_reviewed_N.md)
```

---

## 참고사항

- 원본 파일은 절대 수정되지 않습니다 (새 파일로 저장)
- 라운드마다 점수 변화를 보여줘서 개선 과정을 추적할 수 있습니다
- 나중에 `report-reviewer.md`를 직접 편집해서 리뷰어를 추가/변경할 수 있습니다
