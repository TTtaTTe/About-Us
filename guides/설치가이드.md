# 📦 다관점 문서 리뷰 에이전트 — 설치 가이드

## 설치 방법

### 방법 1: 마켓플레이스 설치 (권장)

Claude Code를 열고 아래 두 줄만 입력하세요:

```
/plugin marketplace add TTtaTTe/About-Us
/plugin install report-reviewer
```

설치 확인:
```
/plugin
```
목록에 `report-reviewer`가 보이면 완료!

업데이트:
```
/plugin marketplace update about-us
```

삭제:
```
/plugin uninstall report-reviewer
```

### 방법 2: 수동 설치

1. `plugins/report-reviewer/agents/report-reviewer.md` 파일 다운로드

2. 아래 폴더에 복사:

| OS | 경로 |
|----|------|
| Windows | `C:\Users\내이름\.claude\agents\` |
| Mac | `~/.claude/agents/` |

> 💡 `.claude` 폴더가 안 보이면?
> - Windows: 파일 탐색기 → 보기 → "숨긴 항목" 체크
> - Mac: Finder에서 `Cmd + Shift + .` 누르기

3. Claude Code에서 `/agents` → 목록에 보이면 완료!

```

```

## 새 플러그인 추가하기

나중에 기획서/기술문서/마케팅 자료용 플러그인을 추가하려면:

### 1단계: 폴더 생성
```
plugins/
└── proposal-reviewer/           ← 새 플러그인
    ├── .claude-plugin/
    │   └── plugin.json
    └── agents/
        └── proposal-reviewer.md
```

### 2단계: plugin.json 작성
`report-reviewer`의 plugin.json을 복사해서 이름과 설명만 변경

### 3단계: marketplace.json에 등록
`.claude-plugin/marketplace.json`의 `plugins` 배열에 새 항목 추가

### 4단계: GitHub에 push
```
git add .
git commit -m "기획서 리뷰 플러그인 추가"
git push
```

이후 사용자들은 이렇게 설치:
```
/plugin marketplace update about-us
/plugin install proposal-reviewer
```

## 트러블슈팅

| 증상 | 해결 |
|------|------|
| 마켓플레이스 추가 실패 | GitHub에 파일이 올라가 있는지 확인. 빈 레포면 실패합니다 |
| 플러그인 설치 후 안 보임 | Claude Code 재시작, 또는 `/agents` 입력하여 확인 |
| 인증 오류 | `gh auth status`로 GitHub 로그인 상태 확인 |
| 업데이트 안 됨 | `/plugin marketplace update about-us` 실행 |
